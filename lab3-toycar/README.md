# JavaScript Lab 3 – Toy Car Object

This project was created for Lab 3 and demonstrates how to use a JavaScript object constructor to represent a toy car product.

The toy car used in this example is based on a product from ModelToyCars.com.

## Features

- JavaScript constructor for creating ToyCar objects
- 13 object properties describing the product
- HTML interface to display the toy car
- CSS styling for the page layout
- Interactive buttons to:
  - display toy car details
  - apply a price discount

## Technologies Used

- HTML
- CSS
- JavaScript
- Git / GitHub

## How It Works

The `ToyCar` constructor allows a programmer to create toy car objects with various properties such as brand, model, color, scale, price, material, and packaging.

An example object is created using the constructor and displayed on the webpage. Buttons allow users to interact with the object and modify some of its properties.
